# DC Motor CAD Model

A detailed **3D CAD model of a DC Motor** designed using **SolidWorks** as part of a Mechanical Engineering design project. The project focuses on modelling individual components, creating a complete assembly, and generating engineering drawings for manufacturing and visualization.

---

## 📌 Project Overview

This project involves designing a complete DC motor assembly in SolidWorks by modelling each component individually and assembling them into a functional model. Engineering drawings were also created for documentation purposes.

The project demonstrates proficiency in:

- 3D Part Modelling
- Assembly Design
- Engineering Drawings
- Mechanical Design Principles
- CAD Documentation

## 📖 Design Workflow

1. Create individual part models in SolidWorks.
2. Apply appropriate dimensions and constraints.
3. Assemble all components.
4. Verify fit and alignment.
5. Generate engineering drawings.
6. Export drawings as PDF.
